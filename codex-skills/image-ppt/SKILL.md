# Image PPT

Use this skill for `图片ppt`, `整页生图 PPT`, visual proposal decks, activity proposal pages, or customer-facing decks where visual impact carries the story.

Rules:

1. Project manager prepares page logic, in-image text, prompts, and acceptance points.
2. Image generation creates full-page images with text inside.
3. User reviews images.
4. PPT Factory assembles confirmed images.
5. Do not use PPT text boxes to fix generated slide text.
