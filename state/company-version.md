# 公司实时版本号

## 当前版本

`TEMPLATE v1.0.0`

## 版本日期

2026-07-02

## 当前版本摘要

这是公开/共享模板初始版。

包含：

- 总管入口规则
- 项目经理边界
- 生图部门硬规则
- PPT Factory 装配规则
- 任务总览板
- 空项目模板
- 最小测试脚本

不包含：

- 真实客户项目
- Notion 镜像
- 运行日志
- 图片/PPT/PDF/视频交付物
- 个人本机路径
- API Key 或账号信息

## 当前硬规则

1. 本地文件是真源。
2. 入口句只负责恢复上下文，不算主任务。
3. 项目经理只做文字、页纲、图内文字、提示词、派单包和验收点。
4. 最终图片必须由生图模型直接生成完整画面和图内文字。
5. PPT Factory 只装配用户确认过的整页图。
6. 没有新鲜验证证据，不得声称完成。
