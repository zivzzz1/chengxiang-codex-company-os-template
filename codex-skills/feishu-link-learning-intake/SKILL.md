# Feishu Link Learning Intake

Use this skill for external learning links.

Rules:

1. Full-text reading is required before claiming learned.
2. Login shells, sidebars, titles, or fragments do not count as full reading.
3. Summarize value, reuse conditions, risks, and next actions.
4. If logged-in browser access is needed but unavailable, say so.
5. Do not pretend inaccessible content was read.
