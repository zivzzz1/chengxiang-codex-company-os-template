# Transcript Learning Summary

Use this skill after transcription or when the user provides long learning material.

Rules:

1. Split long content into chapters.
2. Extract the useful judgment, not just a summary.
3. Identify what can be reused.
4. Identify gaps and unclear parts.
5. Give next actions.
