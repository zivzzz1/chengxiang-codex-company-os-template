# Structured PPT

Use this skill when the main challenge is structure, business logic, sales clarity, or a client-readable deck.

Rules:

1. Build a claim spine before slide production.
2. Define page rhythm and page roles.
3. Write readable slide copy first.
4. Route visual slide production through image/PPT gates when needed.
5. Do not let project managers directly export PPTX/PDF unless the workflow explicitly routes to PPT Factory.
