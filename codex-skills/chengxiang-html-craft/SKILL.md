# Chengxiang HTML Craft

Use this skill for Chinese HTML pages that need clear hierarchy, taste, and public-facing language.

Rules:

1. Do a Design Read before writing the page.
2. Manage attention before writing dense content.
3. Prefer clear sections, visual hierarchy, and decision-oriented summaries.
4. Avoid template-like, card-heavy, internal-looking pages.
5. Verify readability across desktop and mobile when producing an artifact.
