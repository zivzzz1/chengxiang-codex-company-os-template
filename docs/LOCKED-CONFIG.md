# 锁定配置说明

这个文件用于人工检查模板是否已经具备最小可共享配置。

## 回复格式

实质任务开始时固定输出：

1. 任务定型表。
2. 执行前置声明。
3. 下一步只做。

完成、发布、同步或公开前固定输出：

1. 完成审计表。
2. 新鲜验证证据。
3. 未覆盖风险。

## 自动线程标题

纯入口句只做恢复，不作为标题。

| 场景 | 标题 |
| --- | --- |
| 总管恢复 | `总管｜状态恢复` |
| 项目恢复 | `项目名｜项目恢复` |
| 生图部门恢复 | `项目名｜生图队列恢复` |
| PPT Factory 恢复 | `项目名｜PPT装配恢复` |
| 实质任务 | `对象｜任务` |

如果重命名工具需要 `threadId`，先读取 `CODEX_THREAD_ID`。

## 功能调用

- 入口恢复只读本地真源。
- 项目经理只做文字、页纲、图内文字、提示词、派单包和验收点。
- 生图部门只做生图模型直出或编辑。
- PPT Factory 只装配确认后的整页图。
- 记录、同步、发布、公开仓库等外部状态变更，执行前必须说明风险闸门，执行后必须回读验证。

## 发布前检查

```bash
python3 scripts/check_template.py
git status --short
```
