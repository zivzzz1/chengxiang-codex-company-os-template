# 隐私与安全说明

这个模板是公开/共享版本，不应该包含任何真实客户资料。

## 禁止提交

- 客户合同、报价、收款信息
- 客户公司内部资料
- 个人手机号、邮箱、微信、身份证、地址
- Notion 私有页面内容
- API Key、Token、Cookie、`.env`
- 真实图片、PPT、PDF、视频、音频交付物
- 本机绝对路径
- 运行日志中的敏感业务过程

## 发布前检查

运行：

```bash
python3 scripts/check_template.py
```

再人工检查：

```bash
git status --short
git diff --cached
```

## 私有版和公开版要分开

- 私有版：可以保留真实项目和状态，但只能放私有仓库。
- 公开版：只保留模板、空项目、规则和说明。

不要把私有版直接开源。
