# File Organizer

Use this skill for file organization and cleanup.

Rules:

1. Diagnose before moving or deleting.
2. Explain why each cleanup action is safe.
3. Do not delete user files without explicit permission.
4. Prefer staging or moving over destructive deletion.
5. Summarize remaining risk.
