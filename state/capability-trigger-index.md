# 能力触发表

这是公开模板的能力触发总索引。它决定“用户说什么、任务是什么、应该触发什么能力”。

## CAP-TRG-0029｜技能路由前置锁定

- 触发口令：`必须调用技能`、`触发什么技能`、`技能没有触发`、`锁死技能`、`调用一图看懂`、`调用 HTML 汇报技能`
- 自动触发条件：非极小单点任务进入实质推进前；用户点名 Skill / 能力 / 部门；任务交付物命中强制 Skill / workflow / Gate；完成前需要声明已完成、可交付、已固化或已同步
- 输出：`技能路由锁定`
- 入口：`companies/MKT-marketing-consulting/workflows/capability-trigger-visual-audit-v1.md`

## CAP-TRG-0028｜能力触发可视审计

- 触发口令：`触发什么能力`、`用了什么能力`、`能力审计`、`过程可视化`
- 自动触发条件：重要任务定型后；客户可见、视觉、PPT、HTML、图片、视频、报价、合同、同步、发布、系统规则建设
- 输出：能力触发预判面板、过程状态条、完成审计表
- 入口：`companies/MKT-marketing-consulting/workflows/capability-trigger-visual-audit-v1.md`

## CAP-TRG-0025｜入口实质任务定型与自动标题

- 触发口令：`启动总管`、`进入项目`、`启动生图部门`、`启动 PPT Factory`、新任务、继续推进
- 输出：任务定型表、恢复类标题或真实任务标题
- 入口：`companies/MKT-marketing-consulting/workflows/entry-first-turn-task-framing-v1.md`

## CAP-TRG-0024｜HTML 可视化汇报

- 触发口令：`汇报`、`复盘`、`项目阶段判断`、`让我直观看到`、`HTML 汇报`、`可视化`
- 自动触发条件：重要汇报、项目简报、学习总结、测试效果、阶段判断、客户材料预审
- 回话要求：先给结论，再给证据、差距、风险、下一步；需要产物时才生成 HTML 文件
- Skill：`html-visual-reporting`

## CAP-TRG-0027｜字体审美生产规范

- 触发口令：`字体审美`、`字体不好看`、`PPT 字体`、`HTML 字体`、`视频字幕`、`图内文字`
- 自动触发条件：图片、HTML、PPT、视频、海报、KV、信息图、客户可见页面涉及字体或版式
- 输出：字体用途、标题气质、正文 / 字幕气质、字重、字距、留白、颜色对比、可读性验收点

## CAP-TRG-0023｜搜索 / 参考先行

- 触发口令：`搜索`、`找参考`、`行业参考`、`竞品参考`、`审美参考`、`空间参考`
- 自动触发条件：项目经理准备策划、页纲、视觉参考、客户可见方案页、生图派单包或 PPT Factory 派单包，且需要外部公开依据
- 输出：公开来源、可借鉴点、不可照搬点、转译判断

## CAP-TRG-0016｜Goal / Loop / State 长任务控制

- 触发口令：`长任务`、`继续推进`、`循环优化`、`做成流程`、`下一轮入口`
- 自动触发条件：多轮推进、自动化循环、派单包生成或状态恢复
- 输出：可验收 goal、当前 state、验收标准、下一轮入口

## CAP-TRG-0017｜简单分析澄清

- 触发口令：`简单分析一下`、`看看这个`、`判断一下`、`值不值得做`
- 自动触发条件：目标、材料、判断维度或交付形态不清楚
- 输出：分析目标、材料缺口、判断维度、第一轮谨慎结论、下一步动作
